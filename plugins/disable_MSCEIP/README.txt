Disable Microsoft Customer Experience Improvement Program

ABOUT
Remove Telemetry reporting for the Microsoft Customer Experience Improvement 
Program (MS CEIP)

	
CONFIGURATION
The following options can be added to config.ini

	DISABLEMSCEIP (Boolean) - Enable or disable MS CEIP
		Y	- Enable (DEFAULT).
		N	- Disable.


NOTE
Disabling the plugin does not undo changes made in previous runs.
